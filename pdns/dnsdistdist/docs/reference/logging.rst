Logging
=======

There are some functions to create log output.

.. function:: errlog(line)

  Writes a error line.

  :param str line: The line to write.


.. function:: warnlog(line)

  Writes a warning line.

  :param str line: The line to write.


.. function:: infolog(line)

  Writes an info line.

  :param str line: The line to write.
